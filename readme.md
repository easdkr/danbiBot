# Danbi - Bot 

## 다양한 챗봇 기능을 가진 채팅 웹 
![db](./public/db.png)

 - 챗봇 활성화 / 비활성화 
 - 사용자 채팅방 출입시 챗봇이 응답
 - 채팅방 내 스케쥴러
 - 대화 패턴 학습
 - 상황별 이미지/ 음성 응답 
### server.js
 - 채팅웹 서버 
### views.js 
 - 채팅 뷰 
### Assets/js
 - front end 기능  
### public 
 - image/ voice 파일 저장
